## Note

- 重复大颗粒抑制
  - 当皮带缓慢停时，颗粒会向前移动，此时 iou 小于设定阈值 0.8，不会被过滤
  - 因此皮带缓慢停时会有几个重复颗粒，应该对整体无太大影响
  - 不能使用 left、right 基本不变，top、bottom 变的方法进行过滤，因为相机的安装对于皮带运行方向可能并非竖直，同时也不适应斜向安装的场景
